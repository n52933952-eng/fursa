#   B2) Inline JSON: entire one-line {"type":"service_account",...} as the value (must start with "{").

#
# Option C — KEY = FIREBASE_SERVICE_ACCOUNT_BASE64  VALUE = base64 of the ENTIRE .json file (one line)
#   PowerShell: [Convert]::ToBase64String([IO.File]::ReadAllBytes("path\to\serviceAccount.json"))
#
# Option D — KEY = FIREBASE_SERVICE_ACCOUNT  VALUE = one-line JSON {...}
#   Same idea as thredtrain: env JSON or backend/firebase-service-account.json (gitignored).
#   If you get invalid_grant / Invalid JWT Signature, the env value is often truncated — use A or C.
